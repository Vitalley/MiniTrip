# MiniTrip
Miniature trip computer for Audi 80/100/200/A6C4 on PIC16F1847

![Front](MiniTrip_Front.jpg)

Main feutures:
- fuel consumption;
- speedometer;
- odometer;
- hour counter;
- on-board voltage;
- fuel level.


![Back](MiniTrip_Back.jpg)


Testing on Audi A6/100 C4 with Digifant ECU ABK motor.
